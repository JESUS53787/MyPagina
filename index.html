Proyectos Academicos

HOTEL BELLA VISTA

Reserva de habitaciones Online
Abril 2025

El sistema es una plataforma web diseñada para que los clientes puedan reservar habitaciones y servicios adicionales de manera rápida y segura desde cualquier lugar. Los usuarios pueden registrarse y acceder a su cuenta mediante Face ID, lo que garantiza un inicio de sesión ágil y seguro sin necesidad de recordar contraseñas.

las funcionalidades principales se incluyen:

  • Reserva de habitaciones: el usuario puede seleccionar fechas, tipo de habitación y número de huéspedes.
  • Servicios adicionales: permite añadir servicios como desayuno, transporte, spa, tours u otros paquetes especiales.
  • Gestión de perfil: los clientes pueden actualizar su información personal, ver sus reservas y realizar cancelaciones o modificaciones.
  • Seguridad y autenticación: el inicio de sesión con Face ID asegura un acceso confiable y personalizado.
  • Interfaz intuitiva: diseño amigable y responsivo, accesible desde dispositivos móviles y escritorio.

Este sistema busca mejorar la experiencia del cliente, agilizar la gestión de reservas y ofrecer un servicio moderno, seguro y eficiente para hoteles que desean digitalizar sus operaciones.

PETS REFUGE

Software Adopcion de Mascota
Noviembre 2025

Pets Refuge es un software de gestión de adopción de mascotas que permite a los usuarios registrarse y acceder mediante usuario y contraseña para gestionar de manera segura sus adopciones. Cada cliente puede adoptar entre 1 a 4 mascotas, y el sistema asegura que toda la información relevante del usuario esté registrada, incluyendo ubicación y empleo, para facilitar un seguimiento adecuado y responsable de las mascotas adoptadas.
